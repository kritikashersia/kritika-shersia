# AI Impact on Global Jobs (2010–2025)

## Project Overview
This project analyzes how AI adoption has impacted global job markets between 2010–2025.

## Tools Used
- Python (Data Cleaning & Validation)
- SQL Server (Database & Stored Procedures)
- Power BI (Dashboard & Data Visualization)

## Project Workflow
1. Imported dataset into Python
2. Cleaned & validated data using Pandas
3. Created SQL Server database & tables
4. Inserted cleaned data into SQL Server
5. Connected Power BI to SQL Server
6. Built Executive Dashboard

## Key Insights
- 30% increase in AI-related job roles
- Salary growth higher in high AI intensity industries
- Developed countries show faster AI adoption
